javascript模板引擎

javascript模板引擎作为数据与界面分离工作中的重要角色，下面我们从模板引擎的实现原理以及目前流行的模板引擎，模板引擎的优势来介绍一下

javascript 模板引擎的原理

虽然每个引擎从模板语法、语法解析、变量赋值、字符串拼接的实现方式各有所不同，但关键的渲染原理仍然是动态执行 javascript 字符串。

JavaScript模板引擎的优势

js模板的本质是 变静为动

  • 视图和代码逻辑分离、低耦合
  • 可拓展、可维护、开发效率高
  • 代码可读性

  • juicer(淘宝)

  • Embedded JS Templates
