React踩过的坑

React
  • React

setState延迟

  • 分页操作，点击下一页，第一次点击无效，第二次点击生效

利用setState的回调，等到状态更改成功之后再操作

列表页删除一项后，列表页不能实时刷新

原因：删除操作后，后端可能还没处理好，就重新拉取数据，导致数据没有变化

通过设置延迟

或者将列表缓存到数组中，直接对数组进行操作，删除后无需再次拉取数据
